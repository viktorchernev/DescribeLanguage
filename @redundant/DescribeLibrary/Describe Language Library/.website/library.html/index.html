Describe Language Library


The Describe Language Library, or simply "the library" is a large and evergrowing collection of Describe source files, arranged by topics. They are released under the MIT license, so you can use them in your projects, as you please.

The library is available in few different file formats. You can browse it in the original .ds format, but most browsers will simply download the .ds files and not display them as text files if not explicitly set up to do so. This is why you might want to browse it in .txt format, which will be displayed in any browser just fine. Finally, you can download the files as 7z or Zip archives.
